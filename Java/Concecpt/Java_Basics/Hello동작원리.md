## 기본
클래스
- 필드
- 메소드('함수'라고 표현 X)

~~~java
System.out.println("Hello"); //  분해

out: System의 필드
println: out의 메소드

~~~

### 컴파일하기
~~~terminal
$javac Hello.java
~~~

Hello.java -> 컴파일 -> Hello.class

Hello.class -> Byte code

### JVM으로 실행하기
~~~terminal
$java Hello
~~~
