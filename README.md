# Study Notes Repository

이 저장소는 주제 중심으로 정리합니다. 루트에는 큰 주제 폴더만 두고, 각 폴더 안에서는 시리즈, 개념, 실습, 자산 단위로 정리합니다.

## Top-Level Structure

- `AWS`: AWS 학습 노트와 시리즈 정리
- `Architecture`: DDD, 헥사고날 아키텍처 같은 설계 관점 문서
- `Concurrency`: 동시성 이슈와 해결 방식 정리
- `DB`: 데이터베이스 개념, MongoDB, MySQL, ORM, SQL Mapper
- `Engineering`: 리팩토링, 요구사항 분석, DevOps 관련 문서
- `General`: 범용 메모, 커리어 메모, 도구 정리
- `Java`: 자바 개념, Effective Java, 디자인 패턴, 실습 코드, 프로젝트
- `React`: React 학습 노트
- `Security`: 인증/인가 및 보안 관련 문서
- `Testing`: 테스트 코드 일지와 주제별 정리
- `Web`: HTTP, 웹 기초, Nginx, 스토리지, 성능, 웹 도구

## Organization Rules

1. 새 문서는 먼저 주제를 기준으로 폴더를 고릅니다.
2. 시리즈성 문서는 `Series`, 날짜성 문서는 `DailyNotes` 같은 하위 폴더를 우선 사용합니다.
3. 실습 코드와 문서는 같은 주제 폴더 안에서 `Projects`, `Practice`로 분리합니다.
4. `.DS_Store`, `*.swp`, `out/`, `tmp/`, `*.class`, `*.iml`은 추적하지 않습니다.
