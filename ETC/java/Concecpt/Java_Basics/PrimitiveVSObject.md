# Primitive VS Object
| 특징                  | 기본형(Primitive)       | 객체(Object)           |
|----------------------|----------------------|----------------------|
| 메모리 할당            | 스택(Stack)에 할당       | 힙(Heap)에 할당         |
| 속도                 | 일반적으로 빠름           | 일반적으로 느림          |
| 기본값               | 특정 값 (예: int의 0)   | `null`               |
| 연산                 | 직접 연산 가능           | 메서드와 속성을 통한 연산 |
| 크기                 | 고정된 크기              | 가변적 크기            |
| Null 저장 가능 여부  | 불가능                  | 가능                   |


연산
- 기본형 직접 연산 가능
	- +, -, * 등등
- 객체형 메서드와 속성을 통한 연산만 가능
	- 생각해볼점? 
		- Integer a = 3;
		- Integer b = 3;
		- System.out.println(a+b);
	- 어떻게 직접 연산이 가능할까?
		- autoboxing, unboxing
