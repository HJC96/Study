# 제 1 정규화
- 모든 속성은 반드시 하나의 값을 가져야 한다.
- 값은 ATOMIC 해야한다
	- 다가 속성
	- 복합 속성

# 1정규화 대상
- 다가 속성이 사용된 릴레이션
- 복합 속성이 사용된 릴레이션
- 유사한 속성이 반복된 릴레이션
- 중첩 릴레이션
- 동일 속성이 여러 릴레이션에 사용된 경우
